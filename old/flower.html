Info:
The Flower Pattern Generator is a programmed installation con- structing flower patterns from a set of predetermined graphics, placing each floral element randomly in a building block and con- structing a wider image out of a set of generated blocks. It is inspired and informed by traditional headscarf decoration and production processes and contemporary interior textile de- sign methods. The application that generates the patterns is written in Python, informed by contemporary Jacquard loom software.
